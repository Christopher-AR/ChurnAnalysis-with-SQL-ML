# Customer Churn Analysis Dashboard

An interactive **Customer Churn Analysis Dashboard** built using **Power BI**, **DAX**, **SQL Server**, and **Machine Learning** to analyze customer behavior, identify churn drivers, and predict customers at risk.
This dashboard helps businesses **reduce churn**, **identify high-risk customers**, and **improve retention strategies**.

# Project Objective

The objective of this dashboard is to:

* Analyze customer churn trends
* Identify key churn drivers
* Segment high-risk customers
* Provide churn prediction insights
* Support data-driven retention strategies

# Skills Used

* **Microsoft Power BI**
* **DAX (Data Analysis Expressions)**
* **SQL Server Management Studio (SSMS)**
* **Machine Learning (Churn Prediction)**
* **Data Analysis**
* **Data Visualization**
* **Data Modeling**

# Dataset Features

The dataset includes the following attributes:

* Customer ID
* Gender
* Age Group
* Tenure
* Contract Type
* Payment Method
* Monthly Charges
* Total Revenue
* Services Used
* State
* Churn Category
* Churn Reason
* Churn Status

# Customers At Risk Table

Displays predicted churn customers with:

* Customer ID
* Monthly Charges
* Total Revenue
* Total Refunds
* Number of Referrals

This helps identify **high-value customers likely to churn**.


# Business Value

This dashboard helps businesses:

* Reduce customer churn
* Improve retention strategy
* Identify high-risk customers
* Increase customer lifetime value
* Improve service quality
